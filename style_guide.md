# Style Guide: Five Lives (fivelives.health)

This style guide captures the premium, medical-grade aesthetic of Five Lives.

## Colour Palette

| Name | Hex | Sample | Usage |
| :--- | :--- | :--- | :--- |
| **Calming Dark Blue** | `#091c60` | (Primary) | Headings, Primary CTA, Dark Backgrounds |
| **Healthy Green** | `#009e81` | (Secondary) | Accents, "Healthy" Pillar |
| **Cosy Blue** | `#56a9b2` | (Tertiary) | Teal accents, Icons, Tags |
| **Moving Mauve** | `#775dc5` | (Accent) | Physical Activity pillar, specific CTAs |
| **Off White** | `#fefcf9` | (Background) | Main site background |
| **Paragraph** | `#2d3246` | (Text) | Body text color |
| **Muted Grey** | `#808186` | (Muted) | Sub-text, metadata |

## Typography

- **Primary Font:** `Maisonneueweb` (Modern sans-serif)
- **Secondary Font:** `Maisonneueweb book` (Lighter variant)

### Hierarchy
- **H1:** 56px / 68px (700 weight, 1px spacing)
- **H2:** 44px / 52px (700 weight, 0.5px spacing)
- **H3:** 36px / 46px (700 weight)
- **Body:** 18px / 1.46 (500 weight)

## Components

### Buttons
- **Shape:** Pill-shaped (`border-radius: 50px`)
- **Interaction:** Smooth 0.4s transitions for background/color changes.
- **Primary:** Dark Blue bg, Off White text.

### Cards
- **Corner Radius:** 16px
- **Shadow:** `0 4px 19px rgba(0, 0, 0, 0.1)` (Subtle depth)

## Layout Patterns
- **Max Container:** 1440px
- **Horizontal Padding:** 60px (Desktop)
- **Vertical Section Spacing:** 80px
